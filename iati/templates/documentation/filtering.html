Search

The current API implements a combined search query on the title and description fields of the ActivityResource.

Countries

The current API implements a countries filter

{{ api_url }}/api/v2/activities/?format=json&countries=BB|AO|FR

Joining calls

Calls to the API can be joined

{{ api_url }}/v2/activities/?format=json&query=Food&countries=BB|AO|FR
